Exercicis UT 4

Exercicis JSON

  1. Crea un arxiu json.js que contendrà les diferents funcions de tots aquests exercicis.

  2. Crea a json.js una funció que dins d'ella crei un objecte Javascript i torni aquest objecte. Crea la pàgina objecteJavascript.html. Mostra la informació de l'objecte retornat per la funció a la pàgina utilitzant la notació per punts.

    Ves-hi

  3. Crea a json.js una funció que torni una cadena que contengui un objecte en format JSON. Crea la pàgina objecteJSON.html  i mostra-hi aquesta informació utilitzant la notació per punts.

    Ves-hi

  4. Crea a json.js una funció que torni una cadena que contengui un array d'objectes com les que has definit a l'exercici 3 en notació JSON. Crea la pàgina arrayObjectesJSON.html. Mostra a la pàgina la informació d'aquests objectes en forma de taula. La taula ha de tenir una capçalera mostrant el títol de cada columna i un peu que mostri el total de persones.

    Ves-hi

  5. Crea la pàgina objecteFuncionsJSON.html. Utilitza la notació JSON per crear un objecte de tipus cotxe, amb la marca, el model i la matrícula, i una funció que torni tots aquests atributs concatenats. És possible?

    Ves-hi

  6. Crea a json.js una funció que rebi un paràmetre i torni una cadena amb el JSON de l'objecte o l'array contingut dins del paràmetre. Crea la pàgina stringify.html. Crea un objecte d'un dels tipus creats amb anterioritat, un array de sencers, una matriu quadrada i un array d'objectes. Passa'ls a la funció anterior i mostra el resultat dins de la pàgina.

    Ves-hi
